# ⚡ ◄ Kiire Alustamise Juhend

## 1️⃣ Installatsioon (2 minutit)

```bash
# Minge projekti kausta
cd "c:\Users\user\Documents\NPMM25\HTML CSS JS\My work\Turvamees"

# Installige sõltuvused
npm install

# Käivitage server
npm start
```

**Valmis!** Avage http://localhost:3000 brauseris.

---

## 2️⃣ Kaamerase Käivitamine (1 minut)

1. Klõpsake **"Käivita kaamera"** nuppu
2. Aktsepteerige kaamara juurdepääsu
3. Liigutage kätt silma ette

✅ Kui kuulete helikadu, toimib!

---

## 3️⃣ Heliade Seadistamine (1 minut)

Paremal paneel:

```
🔔 Häire helid
├─ Edasi → Beep / Häire / Kell
├─ Tagasi → Häire / Beep / Kell
└─ Küljeale → Kell / Beep / Häire
```

Reguleerige helitugevust olenevalt oma eelistustest.

---

## 4️⃣ Tundlikkuse Muutmine (30 s)

Slider'iga "Liikumise tundlikkus":

- **Madal** (10%) - Ainult teravad liikumised
- **Keskmine** (30-50%) - Normaalne
- **Kõrge** (80%+) - Väike liikumisegi

---

## 5️⃣ Režiimide Kasutamine (30 s)

```
Normaalne  → Kõik näitajad
🌙 Öine    → Tumedad värvid
🔇 Vaikne  → Ainult visuaalne
```
